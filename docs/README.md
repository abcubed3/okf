# OKF-go Documentation

Welcome to the detailed documentation for the Open Knowledge Format (OKF) Go tool suite.

This directory provides in-depth guides on using the `okf` CLI for various tasks, including harvesting metadata, assembling context for LLMs, and setting up proper authentication for cloud resources.

## Table of Contents

1. [Authentication Guide](authentication.md) - Learn how to authenticate with Google Cloud services like BigQuery and Cloud Spanner.
2. [Harvesting Data](harvesting.md) - Detailed examples of how to extract metadata from databases, Git repositories, and web pages.
3. [Context Assembly & MCP](mcp-assembly.md) - Learn how to serve your OKF knowledge graph to AI agents using the Model Context Protocol (MCP) or compile it directly to XML.

---

**Tip:** Run `okf help` in your terminal for a quick reference of all available commands and flags.
